Project Overview

This project focuses on developing an unsupervised system for detecting and clustering sound events within bioacoustic recordings, specifically targeting marine life vocalizations such as fish and whales, while filtering out anthropogenic noises like boat engines.

Key Features

  • Sound Event Detection
  • Unsupervised Clustering
  • Endpoint Detection
  • Bioacoustic Analysis
